Angular File Saver

An AngularJS service that provides cross-browser compatibility of the HTML5 saveAs()

View on GitHub Download .zip Download .tar.gz

Installation

Using bower

$ bower install angular-file-saver

Using npm

$ npm install angular-file-saver

Dependencies

  • AngularJS
  • FileSaver.js
  • Blob.js

Basic usage

  1. Include the fileSaver module to yor project;
  2. Pass SaveAs service as a dependency;
  3. Invoke SaveAs.download() with following arguments:
    • data - data, represented as an array or a Blob object;
    • filename
    • options - set of options for the Blob constructor (optional parameter if Blob object is passed as first argument)

Example:

  var text = ['Hello world'];
  var options = { type: "text/plain;charset=utf-8" };
  SaveAs.download(text, 'textfile.txt', options);
